Install in 30 seconds

The ERPlora SDK is a normal Python package. One command and you have a full local Hub with hot-reload and a fake DB.

$ pip install erplora-sdk
$ erplora init my-module
$ erplora dev

Your module is now live at http://localhost:8000 with auto-reload.
