API

PropTipoDefaultDescripción
attrsdict{}Atributos HTML extra (incluyendo data-signals de Datastar para el view toggle).
(slot)——Estructura interna libre. Sub-clases: .dt-toolbar, .search, .date-range, .view-toggle, .icon-btn, .table, .table-scroll, .dt-grid, .dt-card, .table-footer, .pagination.

El macro datatable es solo el wrapper externo. El detalle (sort, filtros, paginación, view-toggle) lo orquesta el consumidor con HTMX o Datastar — no es responsabilidad del componente.

{# Inline vanilla-JS sort handler — demo-only. The macro stays declarative; the page wires `data-sort-key` + `data-sort-col` headers to a tiny sorter so visitors can interact with the example. Real apps use HTMX/Datastar. #} {% endblock %}
