API

PropTipoDefaultDescripción
attrsdict{}Atributos HTML extra.
(slot)——Estructura interna compuesta libremente. Sub-clases: .ok-dt-toolbar, .ok-search, .ok-dt-toolbar__spacer, .ok-dt-toolbar__divider, .ok-table, .ok-pagination, .ok-pagination__btn.
