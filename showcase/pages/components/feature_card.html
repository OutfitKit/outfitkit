Feature card

Patrón "feature/module card" para landings, marketplaces y secciones de marketing — icono cuadrado + título + descripción corta. Modelo: Stripe, Linear, Vercel, Material 3.

Composición: usa .feature-card dentro de cualquier .grid-3, .grid-4 o .grid-auto. El icono va en un .icon-box reusable que también funciona standalone.

Icon-box standalone

El .icon-box también vive fuera de las cards — para nav items, list items, stats, etc.

API · macro feature_card()

PropTipoDefaultDescripción
titlestr""Encabezado h3.
subtitlestr""Descripción corta p.
iconstr (iconify name)NoneEj: "lucide:shopping-cart". Si None, usa {% raw %}{% call ... %}{% endraw %} con SVG custom.
icon_kindbrand·success·warn·danger·info·neutral·solid·outline"brand"Color/estilo del icon-box.
icon_sizesm·md·lg·xl"md"Tamaño del icon-box.
hrefstrNoneSi presente, renderiza como <a> + activa .interactive.
variantcompact·horizontal·elevated·flat·glassNoneModificador layout/look.

Modificadores CSS de .icon-box

  • Tamaño: .sm, (default), .lg, .xl
  • Kind: .success, .warn, .danger, .info, .neutral, .solid, .outline
  • Forma: .round (círculo)
