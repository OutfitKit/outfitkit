Sparkline

Mini gráfica SVG de línea. Acepta un string de puntos "x1,y1 x2,y2 ..." o una lista de tuplas [(x, y), ...]. Tres tamaños y direcciones up/down/brand.

{% call demo_frame("Default") %} {{ sparkline("0,18 10,15 20,16 30,12 40,14 50,10 60,11 70,7 80,9") }} {% endcall %} {% call demo_frame("Direcciones") %} {{ sparkline("0,18 10,16 20,14 30,12 40,10 50,9 60,7 70,5 80,3", direction="up") }} {{ sparkline("0,4 10,6 20,5 30,9 40,11 50,14 60,15 70,18 80,19", direction="down") }} {{ sparkline("0,12 10,8 20,14 30,9 40,15 50,7 60,12 70,6 80,10", direction="brand") }} {% endcall %} {% call demo_frame("Tamaños") %} {{ sparkline("0,15 14,10 28,12 42,7 56,5", size="sm") }} {{ sparkline("0,18 10,15 20,16 30,12 40,14 50,10 60,11 70,7 80,9") }} {{ sparkline("0,28 12,24 24,26 36,20 48,22 60,16 72,18 84,12 96,14 108,8 120,4", direction="up", size="lg") }} {% endcall %} {% call demo_frame("Lista de tuplas") %} {{ sparkline([(0, 18), (20, 14), (40, 10), (60, 6), (80, 2)], direction="up") }} {% endcall %}
